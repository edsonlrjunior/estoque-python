# Sistema de Estoque em Python
Sistema simples de controle de estoque desenvolvido em **Python**, executado via terminal (CLI), com persistência automática em arquivo **JSON** e suporte para importação e exportação de dados em **CSV**.
Ideal para pequenos negócios, uso acadêmico ou aprendizado de lógica de programação e manipulação de arquivos.
---
# Objetivo do Projeto
Este projeto tem como finalidade oferecer um sistema leve e funcional para:
- Controle de entrada e saída de produtos  
- Monitoramento de estoque mínimo  
- Organização básica de inventário  
- Exportação de relatórios em CSV  
- Importação em lote de produtos  
Tudo isso sem necessidade de banco de dados externo ou bibliotecas adicionais.
---
# Requisitos
- Python **3.8 ou superior**
- Sistema operacional: Windows, Linux ou macOS
- Não é necessário instalar bibliotecas externas (usa apenas bibliotecas padrão do Python)
